This is a simple console game in Python.
The player is represented by [U] on a 10Ã—10 grid.
There are four types of items to collect: [C], [A], [D], [H].
Move the player using w (up), s (down), a (left), and d (right).
Press q to quit the game.
Each time you collect an item, the count of that item is shown.
The board updates after every move.
You can play again after finishing a game.
An optional cat.txt file can show a  cat at the start.
Python 3 is required to run this game.
